# Postal Codes JP
日本の郵便番号を検索するための Web API を提供します。  
Provides the Web API to search postal codes of Japan.

### Development Environment
- .NET Core 2.0
- ASP.NET Core Web API 2.0.8
- Microsoft.AspNetCore.Cors 2.0.3
- Swashbuckle.AspNetCore 2.5.0
